## Logger
- Set environmental variable 'AUTHFUL_LOG_LEVEL' to one of the following levels. Each level will log the levels above it (WARN logs WARN, ERROR, and FATAL)
  - OFF
  - FATAL
  - ERROR
  - WARN
  - INFO
  - DEBUG
  - VERBOSE
  - ALL
